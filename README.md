# Football Player Finder

A client-side web app for searching football (soccer) player profiles. Type any player name and instantly get their team, nationality, position, date of birth, and bio — pulled live from [TheSportsDB](https://www.thesportsdb.com/) free API.

## Features

- Search any player by name with full or partial matching
- Displays team, nationality, position, date of birth, and biography
- Player photo with silhouette fallback when no image is available
- Enter key support and inline error messages
- No backend required — runs entirely in the browser

## Tech Stack

| Layer     | Technology               |
|-----------|--------------------------|
| Structure | HTML5                    |
| Styling   | CSS3                     |
| Logic     | Vanilla JavaScript (ES6) |
| Data      | TheSportsDB free API     |

## Project Structure

```
football player finder/
├── index.html   # Search UI and player card layout
├── script.js    # API fetch, DOM rendering, error handling
└── style.css    # Card design and responsive layout
```

## Getting Started

No install or API key needed. Just open `football player finder/index.html` in a browser.

For best results use full names, e.g. `Kevin De Bruyne`, `Lamine Yamal`.
